# Single Page CV

This project is a solution to the **Single Page CV** challenge from roadmap.sh.

## Description

A simple one-page CV built using semantic HTML.
The goal of this project is to practice proper HTML structure, semantic tags, and basic SEO meta tags.

## Technologies Used

* HTML5

## How to Run

1. Clone the repository
2. Open the `index.html` file in your browser
